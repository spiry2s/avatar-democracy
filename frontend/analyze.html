About this tool

This is the first concrete tool from a broader proposal called Avatar Democracy — the idea that the lower legislative chamber could be replaced with AI agents that vote for each citizen, executing their delegated political will.

The bill analyzer works independently of that vision. It works with current legislatures, current democracy, current institutions. It just makes legislation legible to citizens. If the broader vision never happens, this tool still has value.

If you'd like to read the full proposal, see the white paper in the repo. If you'd like to contribute — better prompts, better UI, better bug reports — find the project on GitHub.

Status: alpha. Expect rough edges. Found a problem? Tell us.
